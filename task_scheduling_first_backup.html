Lorsque vous configurez SaaS Backup pour Microsoft 365, vos données ne sont pas protégées par défaut. Vous devez transférer vos données du niveau non protégé vers l'un des niveaux protégés pour que la sauvegarde de vos données soit effectuée au cours de la prochaine sauvegarde planifiée du niveau sélectionné.

Étapes

  1. Dans le Tableau de bord, sélectionnez le service contenant les données non protégées.

  2. Cliquez sur View en regard du nombre de boîtes aux lettres, de sites Mysites, de sites ou de groupes non protégés.

  3. Sélectionnez les éléments à protéger.

  4. Cliquez sur le menu groupes.

  5. Sélectionnez Tier pour la stratégie de sauvegarde que vous souhaitez attribuer. Voir "Politiques de sauvegarde" pour une description des niveaux de stratégie de sauvegarde.

  6. Cliquez sur appliquer.
